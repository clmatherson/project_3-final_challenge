Transfer Learning and ml5
The page dealing with the training of signs and the page dealing with the self trained video model were completed with the help of p5 https://p5js.org/ and ml5 https://ml5js.org/.

Transfer Learning

Transfer learning is a machine learning method or technique where data learned from one process is transfered or used in a different but similar process. So in our case we used a trained model called MobilNet which was trained to recognize several different images, but not the ones we required, and have allowed the user to retrain it to their signs. We have also trained a second model to recognize the signs we trained it on.

Learning Process

This was a new program to us, that we learned about after we chose our goal and had a deadline. This meant we had to learn on the fly. The program is set up to do a lot of what was done, however at this time in my limited experience it does not connect well with html, bootstrap, or d3, the formating and action driving systems we have generally used in the past. As I am newer to this process, I was not expecting to have issues with moving the location of buttons.
Yes, I did eventually find libraries and code that are supposed to help with actions of buttons and placement of buttons, but it was near the deadline and I ran out of time to figure out how to implement them correctly. I could place the buttons but they were stagnet and did not dynamically move when the page re-sized.
My other big learning curve was in having to find the correct documentation for how things work. There were tutorials, information on how to get started, and information on machine learning and transfer learning all of which was extremely helpful. The information about deploying to a website needing to be set up as a local http sever deploy; especially if you want to be able to upload save models, that part I missed until 1 day before our deadline. These learning areas are just part of the process and means next time there will be more awarness of what needs to be looked into when working with a new program.
