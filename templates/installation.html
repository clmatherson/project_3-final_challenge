Methodology

For our project we looked into building an application would convert ASL hands signs, into the corresponding words. We leveraged Tensorflow for the main component of the application. The high level flow can be found below.

  1. Setup paths
  2. Create label map
  3. Create Tensorflow records
  4. Test then train the model
  5. Detect ASL in real time
  6. Have fun

Acquire Pictures

With finishing the Pipeline into Tensorflow, the next step was to feed it images. Using Images we took, the next step was to feed it into Tensorflow. Using a Jupyter Notebook we recorded us taking the images then using label maker to tell the Algorithm where to look for the ASL hand signs.

  1. Use the Jupyter Notebook to record our ASL handsigns
  2. The Notebook would then label the picture files according to what we were signing
  3. Move the pictures in to a label maker to crop the photos to tell the Algorithm where to look
  4. Transfer the cropped pictures into XML files
  5. Move both into Test and Train folder respectively
  6. Push up to transfer into the Learning Algorithm
